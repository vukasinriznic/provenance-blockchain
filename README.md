# ⛓️ Provenance Blockchain — Landing Page

A sleek, animation-rich single-page website for a blockchain platform, designed and built in **Webflow**.

---

## 📋 Overview

A high-quality, one-page landing page built for a blockchain product. The site focuses on delivering a polished, premium feel through carefully crafted animations, smooth scroll interactions, and a modern visual identity.

---

## ✨ Features

- Single-page layout with smooth scrolling
- High-quality entrance and scroll-triggered **animations**
- Clean, modern UI with strong typographic hierarchy
- Fully responsive across devices
- Fast load time and optimized assets
- Professional-grade interaction design

---

## 🎨 Design Highlights

- Dark, premium color palette fitting the blockchain/Web3 aesthetic
- Micro-interactions on UI elements for a polished feel
- Animated hero section
- Fluid section transitions

---

## 🧱 Built With

| Tool | Purpose |
|---|---|
| Webflow | Design, layout, animations, hosting |
| Webflow Interactions | Scroll and entrance animations |
| Custom CSS | Fine-tuned styling where needed |
